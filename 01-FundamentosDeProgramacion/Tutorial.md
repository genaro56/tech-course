# Fundamentos de programación

#### Objetivos y alcance
Esta sesión cubre una corta introducción sobre la programación como una herramienta poderosa e importante en la actualidad. También comenzarán las sesiones prácticas con ayuda de tutoriales guiados por CodeAcademy. 

Se cubrirán algunos fundamentos de programación con actividades prácticas en el lenguaje JavaScript.

## ¿Qué es programar?
Actualmente contamos con dispositivos y aplicaciones sumamente poderosos al alcance de nuestras manos (nuestro teléfono celular, por ejemplo). Detrás de ello, hay un conjunto de instrucciones que los dispositivos pueden seguir para saber qué hacer y cómo ejecutar este conjunto de instrucciones es el programa (o código) con el que cuenta el dispotivo o aplicación.

### ¿Por qué aprender a programar?
Entre las muchas razones que podríamos enlistar para aprender a programar, aquí hay algunas:
- Ventaja competitiva en casi cualquier industria (ver ejemplos anteriores).
- Es una herramienta poderosa para materializar ideas desde cero.
- Generar gran impacto: actualmente casi todo se está diseñando con orientación hacia la tecnología.
- Mayor alcance: por medio de aplicaciones y servicios en línea, es posible alcanzar una audiencia más amplia gracias a la globalización de la tencología.

#### Learn JavaScript: Introduction (3/9) - Tipos de datos
Un tipo de dato engloba detalles sobre la información que contiene un valor. Por ejemplo, en español diríamos que "hola" es una palabra o texto; mientras que diríamos que 7 es un número. De igual manera, JavaScript tiene 4 formas de representar valores:

- Strings: se le llama así al texto literal que puede contener diferentes caracteres (letras, números, etc). Siembre debe de ir entre comillas simples o dobles. Ej. 'Hola mundo'.
- Números: engloban valores numéricos (enteros, decimales, etc.). Ej. 10, 2.5.
- Booleanos: son parecidos a un *switch* de encendido o apagado. Puede tener sólo dos valores: **true** o **false**.
- Null: representa la ausencia de un valor.

#### Learn JavaScript: Introduction (4/9) - Operaciones matemáticas
Javascript puede hacer operaciones matemáticas como cualquier calculadora. Es necesario únicamente conocer la sintaxis para escribir estas operaciones en Javascript.

##### Suma
```javascript
console.log(2 + 8);
```
##### Resta
```javascript
console.log(20-19);
```
##### División
```javascript
console.log(20/2);
```
##### Multiplicación
```javascript
console.log(0.25*100);
```

#### Learn JavaScript: Introduction (5/9) - Propiedades
Los tipos de dato en JavaScript tienen propiedades. Así como nosotros podemos saber que la palabra "hola" tiene 4 letras, los **strings** en JavaScript tienen una propiedad **length** (en español: longitud). Ésta obtiene la longitud en caractéres de un string.

Sigue las instrucciones imprimiendo en consola lo que pide la actividad. Recuerda hacer click en **Run** después de cada paso, y en **Next** para avanzar a la siguiente actividad.

#### Learn JavaScript: Introduction (5/9) - Métodos predefinidos
Ya vimos que los tipos de datos tienen propiedades. Es importante saber que también tienen lo que llamamos **métodos** o **funciones**. Por ejemplo, los strings tienen un método **toUpperCase()**, el cual escribe el string completamente en mayúsculas.

La siguiente línea, imprimiría en la consola el texto "HELLO WORLD".
```javascript
console.log('Hello World'.toUpperCase()); 
```

```
HELLO WORLD
```

Notarás que los métodos tienen paréntesis a la derecha, a diferencia de las propiedades. **La diferencia entre un método y una propiedad, es que la propiedad es algo que el valor tiene (longitud), mientras que el método es algo que el valor puede hacer (transformar a mayúsculas)**.
